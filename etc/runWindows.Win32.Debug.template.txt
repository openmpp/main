oms web-service:

Template to run Win32 Debug modelNameD.exe on Windows

Rename this file into:
  run.Win32.Debug.template.txt
to make it visible to oms web-service an UI

Oms web-service process this template as:
  - substitute $arguments
  - combine result into single line
  - skip empty lines
Result of template processing is a command line to run model.exe

Example of result:
  ./modelNameD -OpenM.LogToFile true

Arguments of template:
struct {
  ModelName string            // model name
  ExeStem   string            // base part of model exe name, usually modelName
  Dir       string            // work directory to run the model
  BinDir    string            // bin directory where model exe is located
  MpiNp     int               // number of MPI processes
  Args      []string          // model command line arguments
  Env       map[string]string // environment variables to run the model
}

Example of custom template:

cmd
/C
(dir /w) && (set MODEL_FILES="/ab/cd") &&
{{.ExeStem}}D
{{range .Args}}
{{.}}
{{end}}

Example of result from custom template above:
  cmd /C (dir /w) && (set MODEL_FILES="/ab/cd") && modelNameD -OpenM.LogToFile true

*/}}
